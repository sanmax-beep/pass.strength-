# pass.strength

A simple password strength analyzer for Termux and Linux.

## Features
- Offline password analysis
- No password storage
- Strength score and suggestions
- Safe and ethical

## Usage
```bash
pass.strength

git clone https://github.com/sanmax-beep/pass.strength-.git
cd pass.strength-
chmod +x pass.strength
cp pass.strength $PREFIX/bin/
pass.strength
